# 班主任寄语与倒计时软件使用说明

## 软件功能
1. 显示班主任寄语
2. 设置目标日期并显示倒计时天数（如中考倒计时）
3. 半透明效果，返回桌面即可看到
4. 可拖动窗口位置
5. 配置自动保存

## 安装和运行步骤

### 方法一：使用批处理文件（推荐）
1. 双击运行 `setup.bat` 文件
2. 脚本会检查是否已安装 .NET 6.0 SDK
3. 如果未安装，会自动打开 .NET 6.0 SDK 下载页面
4. 请下载并安装 .NET 6.0 SDK，然后再次运行 `setup.bat`
5. 安装完成后，脚本会自动构建和运行软件

### 方法二：手动安装 .NET SDK 后运行
1. 访问 https://dotnet.microsoft.com/zh-cn/download/dotnet/6.0 下载并安装 .NET 6.0 SDK
2. 打开命令行，进入项目目录
3. 运行 `dotnet build` 命令构建项目
4. 运行 `dotnet run` 命令启动软件

## 使用方法
1. **设置寄语**：点击"设置"按钮，在弹出的窗口中输入班主任寄语，然后点击"保存"
2. **设置倒计时**：在设置窗口中选择目标日期（如中考日期），然后点击"保存"
3. **调整窗口位置**：拖动窗口到桌面任意位置
4. **关闭软件**：点击"关闭"按钮

## 技术特点
- 使用 WPF 框架开发
- 半透明窗口设计，美观且不影响桌面操作
- 实时更新倒计时
- 配置自动保存到本地文件

## 注意事项
- 首次运行需要安装 .NET 6.0 SDK
- 软件会在后台运行，返回桌面即可看到
- 配置会自动保存，下次启动时保持上次的设置

## 常见问题
1. **软件无法启动**：请检查是否已安装 .NET 6.0 SDK
2. **倒计时不更新**：请检查系统时间是否正确
3. **设置不保存**：请确保有足够的权限写入配置文件
